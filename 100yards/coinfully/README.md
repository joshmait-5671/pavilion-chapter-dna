# Coinfully

Homepage prototype for [Coinfully](https://www.coinfully.com) — coin appraisal, authentication, and purchasing for inheritors and investors.

**Tagline:** Knowledgeable. Transparent. Fair.

## What's here

- Landing page with coin scanner placeholder, live metals pricing (via Netlify Functions), and team bios
- Travis AI assistant chat UI
- Netlify serverless functions for coin scanning and metals prices

## Stack

- Static HTML/CSS/JS
- Netlify Functions (serverless)
- Fonts: Inter (Google Fonts)

## Run locally

```bash
npm install
npx netlify dev
```

Or just open `index.html` directly (functions won't work without Netlify CLI).

## Deploy

Hosted on Netlify. Push to `main` to deploy.
